# はじめに

Dormitricityへようこそ！このガイドでは、本サービスの基本機能について説明し、寮の電力使用状況の監視、トレンドの理解、タイムリーなアラートの設定を支援します。

## 登録とサブスクリプション

寮を監視する前に、アカウントを作成し、サブスクリプションを追加する必要があります。

1.  **アカウントの作成**：[サインアップ](/auth/register)ページにアクセスしてください。有効なメールアドレスとパスワードを提供する必要があります。登録が成功すると、自動的にログインされ、ダッシュボードにリダイレクトされます。

2.  **最初のサブスクリプションの追加**：
    *   ダッシュボードで、「+」カードをクリックして新しいサブスクリプションの追加を開始します。
    *   カードが入力フォームに変わります。寮のID（例：`5-202`または`E-505`）を入力してください。これはあなたの部屋のユニークな識別子です。
    *   現在、Dormitricityは、ウェブページ経由で電力残量を確認できる本部キャンパスおよび沙河キャンパスのすべての寮をサポートしています。サブスクリプション設定後24時間経っても寮の電力残量がメインページに表示されない場合は、クローラーリポジトリで[Issueをオープン](https://github.com/KeithCoreDumped/dormitricity-crawler/issues/new)してください。
    *   **「追加」** ボタンをクリックします。IDが有効な場合、新しいサブスクリプションカードがダッシュボードに表示されます。

最大3つの寮をサブスクライブできます。

## ダッシュボードの理解

ダッシュボードでは、サブスクライブしているすべての寮の概要を確認できます。各サブスクリプションは独自のカードに表示されます。

### サブスクリプションカードのコンポーネント

*   **寮ID**
*   **最新の残量**：最後に記録された電力残量（`kWh`）と更新時刻（通常10分ごとに最新の測定値に更新）を表示します。
*   **通知ステータス**：現在のサブスクリプションで通知が有効になっているか、また有効な通知ルールの数を示します。
*   **トレンドを見る** <LineChart className="h-4 w-4 inline-block align-text-bottom ml-1" /> ボタン：電力使用量のトレンドチャートページに移動します。
*   **設定** <Settings className="h-4 w-4 inline-block align-middle ml-1" /> ボタン：通知設定を構成またはテストします。
*   **枯渇予測**：電力がなくなりそうな場合、最近の消費率に基づいて枯渇までの推定時間を表示します。

## 使用トレンドの分析

「トレンド」ページでは、電力消費の詳細を確認できます。

*   **時間範囲の選択**：チャートの上で、過去24時間、7日間、30日間など、異なる時間範囲を選択して使用履歴を表示します。
*   **インタラクティブチャート**：最初の折れ線グラフは、時間経過に伴う寮の電力残量（kWh）をプロットします。チャート上の任意の点にカーソルを合わせると、特定の時点での正確な残量を確認できます。
*   **電力推定**：2番目のチャートは、残りの電力残量から推定された過去の電力消費量（kW）を示します。任意の点にカーソルを合わせると、特定の時点での推定電力を確認できます。
*   **枯渇予測**：電力がなくなりそうな場合、最近の消費率に基づいて枯渇までの推定時間を表示します。
