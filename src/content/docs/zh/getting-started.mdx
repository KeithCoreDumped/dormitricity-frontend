# 入门指南

欢迎使用 Dormitricity！本指南将引导您了解本服务的基本功能，帮助您监控宿舍用电情况、了解用电趋势并设置及时提醒。

## 注册与订阅

在监控宿舍之前，您需要创建一个账户并添加一个订阅。

1.  **创建账户**：访问[注册](/register)页面。您需要提供一个<s>有效的</s>电子邮箱地址和密码。成功注册后，您将自动登录并跳转到仪表盘。

2.  **添加您的第一个订阅**：
    *   在仪表盘上，点击“+”开始添加新的订阅。
    *   该卡片将转变为一个输入表单。输入宿舍号码（例如 `5-202` 或 `E-505`）。这是您房间的唯一标识符。
    *   当前Dormitricity支持本部和沙河的所有能通过网页查剩余电量的宿舍。若设置订阅24小时后，宿舍剩余电量仍未在主页显示，请在crawler仓库[发布issue](https://github.com/KeithCoreDumped/dormitricity-crawler/issues/new)
    *   点击 **“添加”** 按钮。如果 ID 有效，一个新的订阅卡片将出现在您的仪表盘上。

您最多可以订阅三个宿舍。

## 理解仪表盘

仪表盘提供了您所有已订阅宿舍的概览。每个订阅都显示在自己的卡片上。

### 订阅卡片组件

*   宿舍号
*   最新电量：显示最近记录的电量余额（`kWh`）和更新时间（通常每10分钟更新最新读数）。
*   通知状态：指示当前订阅是否启用通知，以及启用的通知规则数量。
*   查看趋势<LineChart className="h-4 w-4 inline-block align-text-bottom ml-1" /> 按钮：进入用电趋势图表页面。
*   设置<Settings className="h-4 w-4 inline-block align-middle ml-1" /> 按钮：配置或测试通知设置。
*   耗尽预估：如果电量即将耗尽，页面将根据您最近的消耗速率，显示一个预估的电量耗尽时间。

## 分析用电趋势

“趋势”页面让您可以详细了解您的用电消耗情况。

*   时间范围选择：在图表上方，选择不同的时间范围来查看您的用电历史，例如最近 24 小时、7 天或 30 天。
*   交互式图表：第一个折线图绘制了宿舍电量余额（kWh）随时间的变化。可以将鼠标悬停在图表上的任何点，以查看特定时间的确切余额。
*   功率估计：第二个图表是从宿舍剩余电量估计得到的历史用电功率（kW）。可以将鼠标悬停在图表上的任何点，以查看特定时间的估计功率。
*   耗尽预估：如果电量即将耗尽，页面将根据您最近的消耗速率，显示一个预估的电量耗尽时间。
