
# はじめに

Dormitricity へようこそ！このガイドでは、本サービスの基本的な機能について説明し、寮の電力使用量を監視し、トレンドを理解し、タイムリーなアラートを設定するのに役立ちます。

## 最初のステップ：登録とサブスクリプション

寮を監視する前に、アカウントを作成し、サブスクリプションを追加する必要があります。

1.  **アカウントの作成**：[登録](/auth/register)ページに移動します。有効なメールアドレスとパスワードを入力する必要があります。登録が成功すると、自動的にログインされ、ダッシュボードにリダイレクトされます。

2.  **最初のサブスクリプションの追加**：
    *   ダッシュボードには、**「+」** アイコンが付いた大きなカードが表示されます。これをクリックして、新しいサブスクリプションの追加を開始します。
    *   カードが入力フォームに変わります。寮の **Canonical ID**（例：`5-202` または `E-505`）を入力します。これはあなたの部屋の一意の識別子です。
    *   **「追加」** ボタンをクリックします。ID が有効な場合、新しいサブスクリプションカードがダッシュボードに表示されます。

最大で 3 つの寮をサブスクライブできます。

## ダッシュボードを理解する

ダッシュボードは、サブスクライブしているすべての寮の概要を一目で提供します。各サブスクリプションは独自のカードに表示されます。

### サブスクリプションカードのコンポーネント

*   **寮 ID**：サブスクライブに使用した `canonical_id`。
*   **最新の残量**：最後に記録された電力残量（`kWh`）と読み取り時間（例：「数秒前」）を表示します。
*   **通知ステータス**：現在の通知設定を示すバッジ（例：「オフ」、「オン」、または「低電力」などの特定のルール）。
*   **アクションボタン**：
    *   **トレンドを表示**（チャートアイコン）：その寮の詳細な使用状況トレンドページに移動します。
    *   **設定**（歯車アイコン）：通知設定を構成またはテストするためのダイアログを開きます。

## 使用状況トレンドの分析

「トレンド」ページでは、電力消費の詳細な履歴を確認できます。

*   **インタラクティブチャート**：主な機能は、電力残量（kWh）と時間の関係をプロットした折れ線グラフです。グラフ上の任意の点にカーソルを合わせると、特定の時間の正確な残量を確認できます。
*   **時間範囲セレクター**：グラフの上で、過去 24 時間、7 日間、30 日間など、さまざまな時間範囲を選択して使用履歴を表示できます。
*   **枯渇予測**：十分なデータがある場合、最近の消費率に基づいて、残量がなくなるまでの推定時間が表示されます。
