# 開始

Dormitricity へようこそ！このサービスは寮の電力使用量を追跡し、トレンドを表示し、電力切れを避けるためのタイムリーな通知を受け取るのに役立ちます。

## 仕組み

Dormitricity は、購読された寮の電力残高を定期的（10分ごと）にスクレイピングすることで動作します。このデータは以下を提供するために使用されます：

*   電力残高と電力消費の履歴データ。
*   残高が枯渇する可能性がある時期のトレンドと推定。
*   充電を思い出させる通知。

## 主な機能

*   **ダッシュボード：** 購読しているすべての寮を一目で確認できます。
*   **使用トレンド：** 時間の経過に伴う電力使用の詳細なチャートを表示します。
*   **通知：** 電力が切れる前に警告を受けるためのカスタマイズ可能な通知を設定します。詳細については、[通知ドキュメント](../notification)を参照してください。

## システムアーキテクチャ

システムは、Python ベースのクローラー、Cloudflare Worker バックエンド、および React ベースのフロントエンドで構成されています。
