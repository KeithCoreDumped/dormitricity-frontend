# 如何设置提醒

> ⚠️///施工中///⚠️

Dormitricity 将要支持两类提醒方式：Web hook机器人（企业微信、飞书），Server酱(Serverchan) 提醒API。

## Web hook 群机器人设置

群机器人通过给群聊发送消息，能实现多人的提醒。常用的企业即时通讯APP都提供相似的Webhook接口。

### 企业微信

参考[官方教程](https://open.work.weixin.qq.com/help2/pc/14931)

添加机器人后，将Webhook URL粘贴至Dormitricity配置页面，应用即可

### 飞书

类似

## Server酱

[Server 酱](https://sct.ftqq.com/) 是一个聚合的推送API，这里推荐用于个人推送（服务号消息，不需要建群、拉机器人）
