# 快速开始

欢迎使用 Dormitricity！此服务帮助您跟踪宿舍用电情况、查看趋势，并及时接收通知以避免断电。

## 工作原理

Dormitricity 通过定期（每10分钟）抓取已订阅宿舍的电费余额来工作。这些数据用于为您提供：

*   电费余额和用电量的历史数据。
*   余额可能耗尽时间的趋势和估算。
*   提醒您及时充电的通知。

## 主要功能

*   **总览：** 一目了然地查看所有已订阅的宿舍。
*   **使用趋势：** 查看详细的用电量图表。
*   **通知：** 设置可自定义的通知，在断电前收到提醒。更多详情请参阅[通知文档](../notification)。

## 系统架构

系统由基于 Python 的爬虫、Cloudflare Worker 后端和基于 React 的前端组成。
