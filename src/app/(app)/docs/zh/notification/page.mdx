# 如何设置通知

Dormitricity 支持通过各种渠道发送通知，在您的电费余额较低或预计即将耗尽时提醒您。您可以选择 Webhook 机器人（企业微信、飞书）或 ServerChan 通知 API。

## 配置通知

要设置订阅的通知，请按照以下步骤操作：

1.  **打开设置：** 在总览页面，找到您要配置的宿舍订阅卡片。点击"设置"按钮打开通知设置对话框。

2.  **选择渠道：** 从"通知渠道"下拉菜单中选择您想要的通知渠道。您可以选择"无"、"企业微信"、"飞书"和"Server酱"。

3.  **提供您的令牌或 Webhook 链接：**
    *   **自动检测：** 您可以直接将完整的 Webhook 链接粘贴到"通知令牌 / Webhook 链接"字段中。系统会自动为您检测渠道（企业微信、飞书或 Server酱）。
    *   **手动输入：** 如果您只有令牌或 SendKey，可以将其粘贴到字段中，然后从下拉菜单中手动选择正确的渠道。

4.  **设置通知规则：**
    *   **低电量阈值：** 启用此选项，当您的电费余额低于某个水平时接收通知。您可以设置以 kWh 为单位的阈值。
    *   **耗尽阈值：** 启用此选项，当系统估计您的电量将在指定小时内耗尽时收到通知。

5.  **配置冷却时间：** 从下拉菜单中选择冷却时间。这是通知之间必须经过的最短时间，以防止您收到垃圾邮件。

6.  **保存：** 点击"保存"按钮应用您的设置。

## 渠道特定说明

### 企业微信

按照[官方企业微信教程](https://open.work.weixin.qq.com/help2/pc/14931)创建机器人并获取 Webhook 链接。将链接粘贴到设置对话框中。

### 飞书

与企业微信类似，您可以在飞书群中创建机器人来获取 Webhook 链接。将链接或仅令牌部分粘贴到设置中。

### Server酱

[Server酱](https://sct.ftqq.com/)是一项服务，允许您通过各种个人渠道接收通知。您可以获取 SendKey 并将其用作您的令牌。
